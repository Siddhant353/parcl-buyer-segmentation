# Parcl Buyer Segmentation Project

This project applies machine learning-based buyer segmentation and investment profiling for real estate market intelligence.

## Contents
- Research report
- Streamlit dashboard
- Processed datasets
- Supporting charts
- Analysis notebooks

## How to run the Streamlit app
1. Open terminal in the `Streamlit-App` folder
2. Run:
   `streamlit run app.py`

## Main dashboard data source
`Data-processed/clients_clustered_named.csv`
